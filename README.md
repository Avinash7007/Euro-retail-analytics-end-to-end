# 📊 Euro Retail Analytics | End-to-End Data Pipeline

End-to-end retail analytics pipeline using Python, SQL Server, and Power BI.  
Covers data cleaning, transformation (bronze-silver-gold), and interactive dashboards to analyze sales, profit, and customer behavior.
